# Kanad Bajpai — Student Portfolio

A minimalist, typography-driven developer portfolio built with high-performance HTML, CSS, and vanilla JS. Designed with an editorial brutalist aesthetic to provide a highly optimized, premium browsing experience.

## ✨ Features
- **Fluid Typography:** Built with CSS `clamp()` to perfectly scale text from 4k monitors down to mobile devices.
- **Buttery Smooth Scrolling:** Physics-based scroll mechanics powered by [Lenis](https://lenis.darkroom.engineering/).
- **Custom Cursor:** Hardware-accelerated cursor that inverts colors underneath it using `mix-blend-mode: difference`.
- **GSAP-style Reveals:** Lightweight, high-performance scroll animations built purely with `IntersectionObserver`.
- **Perfect SEO & Accessibility:** Strict semantic HTML5, 100% ARIA compliance, and full Open Graph meta data.

## 🚀 Built With
- Pure HTML5 / CSS3 / Vanilla JavaScript
- Font Awesome
- Lenis Smooth Scroll
- Vite (Build orchestrator)

## 🛠️ Running Locally
This project utilizes [Vite](https://vitejs.dev/) to serve the application instantly and bundle the files cleanly for production.

1. Clone the repository:
   ```bash
   git clone https://github.com/Kanadb30/Portfolio-website.git
   cd Portfolio-website
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the local development server:
   ```bash
   npm run dev
   ```
4. Build the project for production:
   ```bash
   npm run build
   ```
